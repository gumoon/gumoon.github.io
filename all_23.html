laravel的 Elixir 使用

2016/11/26

介绍

Laravel Elixir 提供一个干净、丰富的api为Laravel应用程序来定义 gulp 任务。
它支持css和js的预编译器，像 Sass 和 Webpack。

安装 Elixir

node -v
npm -v
npm install --global gulp-cli
cd project-dir
npm install

运行 Elixir

gulp
gulp --production
gulp watch

处理js

可用做什么？

  1. 编译 ECMAScript 2015
  2. 打包js文件
  3. 最小化js文件
  4. 连接js文件

怎么用？

  1. 配置 Webpack 或 Rollup 编译工具。
  2. laravel的 gulpfile 使用 webpack 来编译js
  3. 使用方法见：https://laravel.com/docs/5.3/elixir

处理css

可用做什么？

  1. 编译 less 文件为css文件
  2. 编译 sass 文件为css文件
  3. 编译 Stylus 文件到css文件
  4. 合并多个css文件
